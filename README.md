# Just Events - Personal Event Planner App
**Student Name:** Sunnie Ho  
**Student ID:** S224863464

## Description
This is an Android application designed to help users manage their daily schedules. It features a minimalist monochrome design.

## Features (CRUD Operations)
* **Create:** Add new events with titles, categories, locations, and dates.
* **Read:** View a list of upcoming events automatically sorted by date.
* **Update:** Edit existing event details.
* **Delete:** Remove events from the local database.
